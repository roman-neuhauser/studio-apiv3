Methods

DELETE

Drop a runner.

Example:

< DELETE /admin/runners/b.runners.susestudio.com
< Accept: $CT:runners

> 204 Ok
> Content-Type: $CT:runners
> Content-Location: /admin/runners
>
> {
>   "a.runners.susestudio.com": { ... }
> , "c.runners.susestudio.com": { ... }
> }
