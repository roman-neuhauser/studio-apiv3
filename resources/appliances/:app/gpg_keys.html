Methods

GET

List GPG keys of an appliance.

Spec:

Accept:
  $CT:gpg-keys

Status:
  200 Ok
Content-type:
  &Accept

Example:

< GET /appliances/123/gpg_keys
< Accept: $CT:gpg-keys

> 200 Ok
> Content-Type: $CT:gpg-keys
>
> {
>   {
>     "name": "testkey"
>   , "target": "rpm"
>   , "key": "-----BEGIN PGP PUBLIC KEY BLOCK-----..."
>   }
> }

POST

Spec:

Accept:
  $CT:gpg-keys
Content-Type:
  $CT:gpg-key

Status:
  200 Ok
Content-type:
  &Accept

Example:

< POST /appliances/123/gpg_keys
< Content-Type: $CT:gpg-key
<
< {
