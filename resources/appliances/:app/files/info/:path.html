Methods

GET

Fetch :path metadata.

Spec:

Accept:
  $CT:file-info

Status:
  200 Ok
Content-Type:
  &Accept

Example:

< GET /appliances/123/files/info/etc/shells
< Accept: $CT:file-info

> 200 Ok
> Content-Type: $CT:file-info
>

> , "group": "users"

  &Accept
