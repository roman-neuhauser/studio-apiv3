Methods

GET

Describe the status of an appliance.

Spec:

Accept:
  $CT:appliance-status

Content-Type:
  &Accept

Example:

< GET /appliances/123/status
< Accept: $CT:appliance-status

> 200 Ok
> Content-Type: $CT:appliance-status
>
> {
>   "state": "error"
> , "issues": [
>     { "type": "error"
>     , "text": "You must include a kernel package in your appliance."
>     , "solution": {
>         "type": "install"
>       , "package": "kernel-default"
>       }
>     }
>   ]
> }
