Methods

DELETE

Delete an appliance version.

This removes all images of the given appliance version.

Spec:

Status:
  204 No Content
| 409 Conflict
| 412 Precondition Failed

Example:

< DELETE /appliance/123

> 204 No Content

Example:

< DELETE /appliance/123

> 409 Conflict
>
> {
>   error: {
>     code: "appliance_is_published"
>   }
> }
